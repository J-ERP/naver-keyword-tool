# 소싱 키워드 선정 기준 v1.0
> 소량다품종 공급사 기준 | 작성일: 2026-06-26

---

## 1단계 — 제외 키워드 필터링

### 정보성 키워드 제거
구매 의도 없이 정보 탐색 목적의 키워드
- 예) 사용법, 후기, 추천, 비교, 원리, 만드는법, ~이란, ~뜻, ~차이, ~하는법

### 브랜드 및 상표권 제거
- 특정 브랜드명, 캐릭터명, 기업 고유명사
- 지식재산권/상표권 침해 우려 키워드

### 부적절 키워드 제거
- 욕설, 비속어, 성인 관련 키워드

### 유아 및 인증 이슈 키워드 제거
- KC인증, 어린이제품 인증 등 추가 규제가 필요한 키워드
- 유아, 어린이, 아동, 키즈, 베이비, 영유아 관련 키워드

## 2단계 — 경쟁도 우선 정렬

> 소량다품종에서는 검색량보다 경쟁도가 더 중요하다.
> 검색량 5만인 HIGH보다 검색량 3천인 LOW가 실제 판매 가능성이 더 높다.

| 우선순위 | 경쟁도 | 이유 |
|---------|--------|------|
| 1순위 | LOW | 신규 상품도 상위 노출 가능 |
| 2순위 | MID | 적정 경쟁, 안정적 판매 |
| 3순위 | HIGH | 대형 브랜드 경쟁, 진입 어려움 |

---

## 3단계 — 동일 경쟁도 내 검색량 정렬

같은 경쟁도 안에서는 검색량 높은 순으로 배치

**예시 (LOW 경쟁도 내)**
- A 키워드: 검색량 12,000 → 1순위
- B 키워드: 검색량 8,000 → 2순위
- C 키워드: 검색량 3,000 → 3순위

**최종 우선순위 요약**
```
LOW + 높은 검색량
↓
MID + 높은 검색량
↓
HIGH + 높은 검색량
```

## 4단계 — 태그 확장 전략

> 실제 판매량은 상품명보다 태그와 노출 키워드 설계에서 많이 갈린다.

### 태그 구성 비율 (총 20개 기준)

| 분류 | 비율 | 개수 | 설명 |
|------|------|------|------|
| 핵심 키워드 | 40% | 8개 | 상품명 직접 연관 키워드 |
| 장소 확장 | 30% | 6개 | 사용 장소 기반 확장 |
| 용도 확장 | 20% | 4개 | 사용 목적/기능 기반 확장 |
| 사용자 확장 | 10% | 2개 | 타깃 사용자 기반 확장 |

### 확장 예시 (욕실 밀대 기준)

**장소 확장 (30%)**
- 가정용 욕실, 카페 화장실, 식당 화장실
- 사무실 화장실, 수영장 샤워실, 관공서 화장실
- 체육관 샤워실, 펜션 욕실, 모텔 욕실

**용도 확장 (20%)**
- 물기 제거, 바닥 청소, 미끄럼 방지
- 물청소, 대청소, 업소 청소

**사용자 확장 (10%)**
- 자영업자, 카페 사장님, 청소업체
- 관리사무소, 건물관리인

### 위험 키워드 예시 (제외 대상)
상품과 카테고리만 비슷하고 구매 전환율이 낮은 키워드
- 욕실 밀대 검색자에게 → 쓰레기통, 세탁바구니, 변기커버 (X)
- 클릭은 생기지만 구매 없음 → 플랫폼 노출 품질 점수 하락

## 5단계 — 확장성 점수 (소싱 의사결정 기준)

> 확장 가능 시장 수가 많을수록 장기 생존 가능성이 높다.

### 점수 기준

| 확장 시장 수 | 확장성 점수 | 판단 |
|------------|------------|------|
| 6개 이상 | 9~10 | 소싱 1순위 |
| 4~5개 | 7~8 | 소싱 검토 |
| 2~3개 | 4~6 | 신중 검토 |
| 1개 | 1~3 | 가격 경쟁 위험 |

### 확장 가능 시장 예시

| 상품 | 검색량 | 경쟁도 | 확장성 | 총점 |
|------|--------|--------|--------|------|
| 욕실 밀대 | 8,000 | LOW | 9 | 92 |
| 변기솔 | 15,000 | MID | 4 | 73 |
| 청소장갑 | 30,000 | HIGH | 3 | 58 |

### 확장 가능 시장 체크리스트
- [ ] 가정용
- [ ] 카페/식당 등 요식업
- [ ] 숙박업 (펜션/모텔/호텔)
- [ ] 관공서/공공시설
- [ ] 체육시설/수영장
- [ ] 사무실/오피스
- [ ] 청소업체/관리업체

## 최종 선정 알고리즘 요약

```
1. 금지 키워드 제거 (정보성/브랜드/욕설/유아)
2. 경쟁도 LOW 우선
3. 같은 경쟁도면 검색량 우선
4. 태그 비율 적용 (핵심40 + 장소30 + 용도20 + 사용자10)
5. 확장성 점수 계산
6. 최종 상위 20개 선정
```
